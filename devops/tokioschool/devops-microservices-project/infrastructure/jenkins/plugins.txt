## Installation via Jenkins CLI

```bash
# Install plugins via CLI
jenkins-cli -s http://localhost:8080/ install-plugin \
  git \
  github \
  docker-plugin \
  docker-workflow \
  workflow-aggregator \
  junit \
  htmlpublisher \
  credentials-binding
```
